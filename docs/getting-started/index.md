## Prerequisites

Before you begin, ensure you have:

- **Python 3.10+** (3.11 recommended)
- **Git** for version control
- **Claude Code** or another MCP-compatible client

!!! warning "Python Version Lock"

    Once you create a Pixeltable database, you must use the same Python minor version.
    See [ADR-001](../adrs/ADR-001-python-version-requirement-for-mcp-servers.md) for details.
